Projeto de autenticação de usuário do desenvolvedor João Mário.

Favor notar que a autenticação via Google não estará disponível em pleno
funcionamento neste projeto pois as novas políticas de uso do GoogleOAuth2.0
exigem comprovação de posse de domínio e outras declarações formais para que
o Google possa aprovar a análise do uso de recurso da API no site, tornando
não prática para aplicação em um projeto que tem finalidade apenas de estudo.
Porém todo o código necessário para este tipo de autenticação  está presente
neste projeto e disponível para acesso no repositório. Não fiz questão de
ocultar as keys do Google ou outros segredos do .env pois a finalidade do
projeto é unicamente didática.
